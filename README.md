# Student Behavior Prediction Model

## Description
This project aims to predict student behavior (Good vs. Bad) based on academic performance and attendance records. Using machine learning, the model analyzes factors like Math, Science, and English grades, alongside absences and behavioral issues, to provide insights into student outcomes.

## Key Features
- **Data Analysis**: Visualizing grade distributions across different subjects.
- **Predictive Modeling**: Using Logistic Regression to classify student behavior.
- **Automation**: Ability to input new student data and receive an immediate behavioral prediction.

## Technologies Used
- **Python**: Core programming language.
- **Pandas**: For data manipulation and CSV handling.
- **NumPy**: For numerical operations.
- **Scikit-Learn**: For building and training the Logistic Regression model.
- **Matplotlib**: For data visualization.
